# Echoing API

Простое FastAPI-приложение, которое возвращает подробную информацию о каждом запросе на единственном эндпоинте `/`.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install fastapi uvicorn
uvicorn main:app --reload
```

## Использование

Отправьте любой HTTP-запрос на `http://localhost:8000/` и получите JSON с:

- методом, схемой, URL и версией HTTP
- заголовками, cookies и query-параметрами (включая дубликаты)
- телом запроса (raw, base64 и попытка декодирования)
- информацией о клиенте и сервере
- сериализованной частью ASGI scope (метаданные соединения)

Приложение удобно для отладки вебхуков, обратных вызовов и тестирования интеграций.
